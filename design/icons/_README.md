# Calendar Game — app icon masters

Two saved candidates (purple-dusk, full-bleed, mask-safe). Both are local design source —
they are NOT in `public/`, so nothing here ships until we wire the chosen one into the PWA (Stage D3).

## ✅ CHOSEN — `icon-piday-trace.svg`

The hand-method **trace** for **Pi Day, 3/14/1592** (a Saturday; π = 3.14159…).
Finger positions **2 → 3 → 6**, landing on **6 = Saturday**. Faint dots = the rest of the
7-position board (0,1,4,5); solid dots = the on-path nodes (2,3); the **circled** dot = the answer.
This is the icon we're using for D3.

## 🗄️ BACKUP — `icon-julian-calendar.svg`

The **1582** Gregorian reform: the calendar jumped **Oct 4 → Oct 15** (the dashed gap = the
dropped days). Kept readily available in case we switch. If it ever becomes the chosen icon,
outline the "4"/"15" `<text>` to paths so rasterization is font-independent.
